All the examples have been built using https://docs.python.org/3/tutorial/index.html
The examples are custom created to make them easier to understand and relatable to real world problems

helloworld.py
------------
First program in Python. Prints "Hello World"

hello_input.py
--------------
Describes how a user input can be captured for processing

string_functions.py
------------------
Shows String manipulation, printing and data typing between different data types

date_simple.py
--------------
Simple program to illustrate Date formatting and use of Dates 

conditional_statements.py
-------------------------
Illustrates the conditional statements and exception handling

loops.py
-------
Describes loops and continue/break from a loop 

functions.py
------------
Describes the logical grouping with functions, advantages of functions
